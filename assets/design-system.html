§ 0.0 · Preface

A voice, not a style.

Every decision below answers one question: what would a quiet, competent gatekeeper look like if we gave it a typeface?

Personality
Unix-native · institutional · dry

Never loud unless
a verdict is loud

Neutral on coding
Emphatic on risk

sir is named after a deferential form of address — the one a butler gives, a bailiff uses, a compiler mimics. That frame earns the whole system. The design is a uniform: understated, monogrammed, always in attendance. It steps forward only when something requires a verdict.

Said differently: it should feel like a receipt, not a dashboard. Like a ledger entry, not a push notification. Like a well-tailored jacket, not a siren.

i Reading this document

Each chapter opens with a single committed recommendation, then presents two or three alternates in smaller type. Pick from the alternates; the primary is the house style.

§ 1.0 · Mark

Four marks for an inverted sandbox.

The current shield-and-arrow doesn't quite land — shields connote defense from outside, not containment turned outward. These explore the actual idea: a boundary drawn around the agent, with authority flowing out only when the lease permits.

Read at 16px

Works in one color

Encodes the model,
not a metaphor

Primary recommendation highlighted below. Others are credible alternates.

Reverse Brackets  · primary

Mk · 01

Brackets turned outward — containment reversed. The three dots are the verdict triad, the mark's semantic core.

Turnstile

Mk · 02

A gate: one direction granted (green, out), one denied (red, in). Directly encodes the lease-boundary mental model.

Taint Trail

Mk · 03

Literal IFC: a secret read, taint propagating along a dashed path, terminating in a denial. The logo is the model.

Butler's Seal

Mk · 04

A wax-seal monogram. Leans into the "sir" wordplay — a stamp of approval that can also say no. Institutional, slightly handmade.

To distinguish verdicts, session posture (tainted / clean), and enforcement layer (hook / IFC / runtime). Always paired with a text label in CLI output — color alone is never load-bearing.

§ 3.0 · Type

A serif for counsel. A mono for conduct.

The house pairing leads with a serif — unusual for a security tool, deliberate here. The serif frames prose as considered; the mono carries verdicts. Three pairings, ranked.

A · Plex Serif + Plex Sans + Plex Mono  · primary

IBM · open

quiet on coding,
loud on denial.

Plex Serif carries prose, titles, and any writing that reads like counsel. Plex Sans holds UI chrome, buttons, and table labels. Plex Mono is the agent's voice — verdicts, paths, hashes, session IDs. All three are drawn by the same team; they fit.

Fraunces softens into handmade territory — good for long threat-model essays, research write-ups, the "why sir exists" page. Newsreader runs cleaner than Plex for body prose on docs pages. Less restrained. Consider only if the docs site becomes the primary surface.

§ 4.0 · Glyphs

Three words. Three shapes. One color each.

Terminals can't animate reliably and don't always render color. The glyph system degrades cleanly: ANSI color drops to shape; shape drops to word. Nothing ever loses meaning.

Label first
glyph second
color last

Never rely
on color alone

For web UI only. High density, high contrast. Unsuitable for terminal.

i Degradation rule

The reason these stack matters. In a TTY with NO_COLOR=1, the color track falls away and the glyph carries; in an extremely constrained terminal, the glyph can fail and the label still carries. The label is the ground truth.

§ 6.2 · Callouts — verdict-rail

· allow

Hooks installed and ledger chain verifies. sir doctor reports nothing to fix. Normal coding is unaffected.

? ask

The agent wants to write a posture file. Posture writes always prompt — this is non-negotiable and documented in CLAUDE.md · §9.

× deny

Session is tainted. A secret read earlier this turn is now preventing outbound network calls, commits, and pushes until the turn boundary resets or the lease widens.

i note

The default lease allows push to origin, commit, loopback, and delegation. Tighten with sir trust or managed policy.

§ 8.0 · Motion

Move only when asked.

Almost nothing animates. The exceptions are three: a pulsing dot while the oracle deliberates, a typewriter for verdicts being printed, and a subtle shake when the system has to shout.

Durations
1.6s pulse
3s type
0.5s shake

Easing
ease-out · step

Reduce
motion respected

Deliberation
ask · pulse 1.6s ease-out inf
ask · waiting for operator…
Verdict print
allow · step 3s inf
· allow · fs.read ~/project/README.md
Emphasis
deny · shake 0.5s on-trigger
× deny · net.egress tainted session
i Rule of thumb

Animation is a verdict too. If it moves without reason, it reads as noise — the opposite of what sir promises. Prefer instant redraws for state, reserve motion for ask and deny.
